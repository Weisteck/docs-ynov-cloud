    Installation et configuration sur la vm Terraform

    Pré-requis:

    • Installer Terraform avec la doc officielle
    • Générer une clef ssh avec la doc GitHub
    • Ajouter sa clef ssh générer à notre compte de GitHub

    Les deux derniers pré-requis vont nous permettre de pouvoir clone notre repot Terraform sur la VM

    Se connecter en ssh sur la machine d'OpenStack afin de tester les étapes d'installation de la documentation officielle de DevStack.

    Ne pas faire la parti add stack user qui est optionnelle
